K-means算法最终的效果受初始中心点的影响， 初始中心点的位置不同，最后分出来的集群也可能不同。 所以K-means算法可能得不到最优解。

那我们可能随机进行多次K-means算法, 然后比较每次划分集群的优劣。

K-means的性能指标叫做惯性， 他是指所有点到所在集群中心点的均方距离。惯性越大，说明模型越差。

我们可以通过inertia_属性访问kmeans的惯性, 而score()则范围负惯性， 因为在Scikit里总是分数越大模型越好。

K-means++算法提出了一种选择初始中心点的改进算法:

  1. 首先随机从数据中选择一个实例c1c^1c1作为中心点.
  2. 按以下概率选择一个新的实例:

    D(xi)2∑j=1mD(xj)2\frac{D(x^i)^2}{\sum_{j=1}^mD(x^j)^2} ∑j=1m​D(xj)2D(xi)2​

    其中D(xi)D(x^i)D(xi)表示实例xix^ixi距离已选择的最远中心点的距离。所以离已选择中心点越远的实例， 越容易被选中。
    注意是容易被选中， 而不是一定会被选中。
  3. 重复第二步, 直到初始化了k个中心点

这个算法保证了初始中心点都尽量离得比较远， 这一改进让K-means算法收敛到次优解的概率很小。

Kmeans类也默认使用这种初始化中心点的方法，如果你一定要随机，可以设置超参数init为random

加速K-Means和小批量K-Means

ELKan提出了一种能加快K-Means算法的方法，利用三角不等式,并通过跟踪示例和中心点之间的上下限距离来减少计算距离的次数，从而加快了K-Means算法。

具体做法是在每次迭代中保存每个点到最近集群中心点的距离， 如果迭代后中心点相对于上一次迭代变化不大，那么就不再计算离这个中心点比较近的实例到这个点的距离。

KMeans类也默认使用这一算法进行加速。除非你要把超参数algorithm设置为null。

K-Means也可以用小批量移动中心点， 而不是使用完整的数据集。 这也让算法的速度提高了3-4倍。

在Scikit-Learn中用MiniBatchKMeans实现了小批量K-Means

不过小批量K-Means的惯性会比标准K-Means的惯性大。

从上图我们可以看到， 随着k的增加，惯性一开始急速下降， 但是在K=4之后下降速度变得缓慢。 整个图像像是一个手肘， 而k=4这个点就是肘部。

那么我们通常可以取肘部这个点作为k值。不过这个方法有点太粗糙了， 所以很少使用。

另一个方法需要更多的计算，但是更好地得出k值。

使用轮廓分数，他是所有实例的平均轮廓系数。

轮廓系数又等于(b−a)/max(a,b)(b-a)/max(a,b)(b−a)/max(a,b), 其中a是实例到同一集群中其他所有实例的平均距离(即集群内的平均距离),b表示该实例到其他集群内所有点的平均距离中， 平均距离最小的那个值。

关于b的描述可能有点抽象，我们结合图来说明一下:

图中的红点属于集群1， a就是他到集群1中其他所有实例的平均距离。然后计算该点到集群2中所有实例的平均距离， 集群3中所有实例的平均距离…最后得到该点到集群2所有实例的平均距离最小， 那么b就是这个值。

轮廓系数是一个(-1, 1)的数

  • 这个值越接近1， 说明这个点在集群中越在中心位置。
  • 这个值越接近0，说明这个点在两个集群的边界附近。
  • 这个值越接近-1， 说明这个点被分配到错误的集群了。

用所有实例的平均轮廓系数来评估一个k值的好坏，就比用肘点判断要靠谱得多了。

在Scikit—Learn中用silhouette_score来计算轮廓分数。

轮廓图里， 每一个像刀一样的轮廓，就是该集群中的实例的轮廓系数排序后组成的。 虚线表示平均轮廓。

当k=6和k=3时， 都有很多实例的轮廓系数小于平均系数， 表示模型糟糕。(k=3时有明显的一个刀刃)。

1.2 K-Means的局限

K-Means有不少优点，能快速训练并且可扩展。但也有不少局限。

  1. 必须要指定集群数量
  2. 需要多次运行算法避开次优解。
  3. 对于椭圆形的集群聚类效果不太好

2. DBSCAN算法


DBSCAN也是一种聚类算法。他的聚类步骤如下:


    

  1. 对于每一个实例， 算法都会计算在距离他ε内有多少个实例，这个区域叫做ε-领域。
    2. 

  2. 如果一个实例在他的ε-领域中有至少min_sample个实例(包含他自己), 那么这个实例就是一个核心实例
    3. 

  3. 核心实例的ε-领域内的实例就会被归为同一集群。他的集群里可能也包含其他核心实例，因此， 集群是包含一连串核心实例的集群。
    4. 

  4. 不属于任何集群的实例，就会被认定为异常。
    5.
