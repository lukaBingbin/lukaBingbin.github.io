机器学习入门以及简单的线性回归模型

一、机器学习入门

1. 什么是机器学习？

机器学习是一门研究在没有明确编码下让计算机获得学习能力的领域。

Field of study that gives computers the ability to learn without being explicitly programmed

在接受一个输入(input)的时候，我们希望机器通过自己学习已有的数据，然后给出一个预测的输出(output)。

这里没有明确编码指的是，没有一套算法告诉计算机如何从输入计算到输出。

我们依然需要一套算法来让计算机学习已有的数据，这样才能预测出一个输出。

2. 机器学习的分类

机器学习可分为监督式机器学习和非监督式机器学习。

2.1 监督式机器学习 (Supervised Machine Learning)

通过已有的正确映射关系(input -> output 或者 x -> y)学习，然后能够接受一个input预测出output。

特点就是学习的数据集里既有输入也有输出。

比如目前有一个深圳市的房价数据集

数据集里输入为房子尺寸(m²)， 输出为房子的价格(10000 CNY)

监督式机器学习就是让机器通过这一组数据进行学习， 最后能够接受任意一个尺寸并给出房价的预测。

监督式机器学习又可以分为 回归 和 分类。

  • 回归 : 根据输入从连续的值中预测出一个值。 比如上面提到的预测房价，或者预测股票价格等。
  • 分类 : 根据输入从有限的值中预测出其中一个值。比如判断一封邮件是否是垃圾邮件。

回归和分类最大的区别就是，回归的预测值可能性是无限的。例如在房价预测中，房价可能是10w到1000w中的任意一个值；而分类的预测值是有限的，从有限的类别中输出一个。

2.2 非监督式机器学习 (Unsupervised Machine Learning)

与监督性学习相反， 非监督性学习的数据集并没有经过标注。也就是说数据集里只有输入， 没有输出。 需要通过算法让机器找到数据本身的内在属性和关系。

例如Google新闻每天都会有上百万个新闻，非监督式机器学习的任务就是找到这些新闻之间的结构并将其聚类。

如图所示，算法能自动从上百万个新闻里聚类出有关熊猫双胞胎、动物园的新闻。

二、 简单的线性回归模型

1. 线性回归的定义

线性回归是用于找到特征值(输入)和结果(输出)之间线性关系的方法。

让我们回到房价的例子中去，例如现在有这样一组数据。

我们希望通过学习后，能够根据房子的任意尺寸预测出这个房子的价格。

所以这里尺寸是特征值(输入)x， 价格是输出y。

把数据集在坐标轴中画出来，如下图所示。

线性回归的任务就是从这些数据中找到一个线性模型，能够预测出任意尺寸房子的价格。

2. 一元线性回归模型

在房价预测的例子中，我们的特征值只有一个尺寸。

我们希望机器在学习了已有的数据之后，能够得到一个线性模型从而进行预测。

从坐标系中可以大概看出，房价和尺寸大概满足一个线性关系

于是我们定义一个预测函数F(x)

F(x)=wx+b=y^F(x) = wx +b =\hat{y} F(x)=wx+b=y^​

其中

y^\hat{y} y^​

表示对于输入x， 给出的预测值。

所以我们的目的就变成了，找到合适的w和b值，使得预测函数F(x)的值更准确。

3.成本函数

既然目的是找到合适的w和b使得模型预测更准确，那么我们就需要评估w和b的成本。

对于上面的例子， 我们假设w = 10, b = 0,于是就得到了上图坐标系中的直线F(x) = 10x。

要评估w和b的成本，就需要和真是的数据做对比。

我们发现在x=60, 75, 100, 140的时候， 模型给出的预测数据都有一定的偏差。

绿色表示实际数据，红色表示预测数据

此时我们可以计算出每一组数据的偏差，为了确保是正数，我们可以将结果进行平方。 再将所有的偏差相加后求平均，就能够得到平均的偏差。

在本例子中就等于:

((f(60)−550)2+(f(70)−800)2+(f(100)−900)2+(f(120)−1200)2+(f(140)−1800)2+(f(200)−2000)2)∗16((f(60)-550)^2 + (f(70) - 800)^2 + (f(100) - 900)^2 + (f(120) - 1200)^2 + (f(140)-1800)^2 + (f(200) - 2000)^2) * \frac{1}{6} ((f(60)−550)2+(f(70)−800)2+(f(100)−900)2+(f(120)−1200)2+(f(140)−1800)2+(f(200)−2000)2)∗61​

得到结果 = 14583.333333333334，这就是F(x) = 10x这个模型的成本。

同理假设w = 9, b = 100， 得到模型F(x) = 9x + 100.

也可以计算得到成本 = 18560.416666666668

也就是说 F(x) = 9x + 100 的效果要比 F(x) = 10x 的成本高， 预测效果更差。

至此我们就可以给出成本函数的式子定义了:

J(w,b)=12m∑i=1m(F(xi)−yi)2=12m∑i=1m(wxi+b−yi)2J(w,b) = \frac{1}{2m} \sum_{i=1}^{m}(F(x^i) - y^i)^2 = \frac{1}{2m} \sum_{i=1}^{m}(wx^i+b-y^i)^2 J(w,b)=2m1​i=1∑m​(F(xi)−yi)2=2m1​i=1∑m​(wxi+b−yi)2

前缀是1/2m是为了后面方便计算。

我们就可以用成本函数来评估一个模型的好坏了。

4.梯度下降

现在我们的目标是找到一对合适的(w,b)值，使得我们的模型函数F(x)预测更精准。而评估一对(w,b)值可以用成本函数来评估。那现在的问题就是如何找到一对最合适的(w,b)值。

4.1 成本函数的图像

我们要找的最合适的(w,b)值，即是使得成本最小的值。如何找到这个值让我们从函数图像来探索。

4.1.1 F(x) = wx的成本函数图像

为了方便理解，我们先简化一元线性模型为F(x) = wx， 暂时舍弃b。

那么此时的成本函数为

J(w)=12m∑i=1m(F(xi)−yi)=12m∑i=1m(wxi−yi)2J(w) = \frac{1}{2m} \sum_{i=1}^{m}(F(x^i) - y^i) = \frac{1}{2m} \sum_{i=1}^{m}(wx^i - y^i)^2 J(w)=2m1​i=1∑m​(F(xi)−yi)=2m1​i=1∑m​(wxi−yi)2

这时候成本函数是一个二次函数，他是图像是一个凹状的。

假设现在我们有一个w1的值，在如图所示的位置

此时的成本是很大的，想要缩小成本，此时w值就需要增大。

或者我们还有一个w2的值，在如图所示的位置

这时候的成本也不是最小的，想要缩小成本，此时w2的值就应该减小。

我们最终需要找到图像中的最低点w3， 这时成本才是最小的。

那么如何确定某一时刻w的值应该减小还是增大呢？我们可以作w点的切线，获取斜率。

当斜率<0时，w应该增大⬆；当斜率<0时，w应该减小⬇。

斜率能够通过求导得到，所以我们可以得到。

w=w−α∂J∂w(其中α为学习率，马上会提到)w = w-\alpha \frac{\partial J}{\partial w} \\ (其中\alpha为学习率，马上会提到) w=w−α∂w∂J​(其中α为学习率，马上会提到)

注意， 这里的"="意为赋值而不是相等

通过这个式子，我们就能知道w应该如何变化才能使成本更小。

4.1.2 F(x) = wx + b的成本函数图像

现在我们尝试把b加回来。这时候的成本函数就应该是

J(w,b)=12m∑i=1m(wxi+b−yi)2J(w,b)=\frac{1}{2m}\sum_{i=1}^{m}(wx^i +b-y^i)^2 J(w,b)=2m1​i=1∑m​(wxi+b−yi)2

此时的成本函数是关于w和b的函数。如果画一个3D图形，那么大概是长这样。

可以看到他同样是一个凹型的图像。

这时候我们要同时变化w和b，也就是

w=w−α∂J∂wb=b−α∂J∂bw = w-\alpha \frac{\partial J}{\partial w} \\ b = b-\alpha \frac{\partial J}{\partial b} w=w−α∂w∂J​b=b−α∂b∂J​

注意w和b是同时变换的，也就是在代码实现的时候，你需要先用两个临时变量保存w和b计算后的结果，再一起赋值。否则在计算b的时候会出错。

如果你在三维空间里从上往下看这个图像，得到的二维图像会是这样的

花一颜色的线上的点，花的成本相同。

4.2 梯度下降

从上一节我们从函数图像看出来了，一组不合理的w和b值应该朝上面方向变化才能使成本更低。

有人会有疑问，我们已经画出来了函数图像，为什么不直接取函数图像中的最低点呢？

这是因为我们这里讨论的只是一元线性回归模型，它的成本函数是可以直观画出来，并且是一个凹形的图像，有一个最低点。

但是当我们的特征值有多个的时候，我们无法直接画出高维度的成本函数图像。有的成本函数图像是一个草地公园，有小山峰也有小低谷。

像这样的函数图像，他有多个局部最低点，并非一个完全的凹形图像。

所以你要做的是，当你站在一个高峰处时，尽可能高效的到达最近的一个低谷处。

所以想象一下，你站在顶峰，环顾四周，选择一个最陡的方向迈下去，然后重复这个操作直到你到达了低谷， 周围的地方都比你所处的位置要高。

最终你可能选择了这样一条路到达了其中一个局部最低点。

这就是梯度下降，当我们有一组(w,b)的值成本过高时，通过不断变化(w,b)的值让成本降低，直到成本无法继续降低或降低幅度很小了。

while(lastJ−J<0.001):w=w−α∂J∂wb=b−α∂J∂bwhile (lastJ -J < 0.001):\\ w = w-\alpha \frac{\partial J}{\partial w} \\ b = b-\alpha \frac{\partial J}{\partial b} while(lastJ−J<0.001):w=w−α∂w∂J​b=b−α∂b∂J​

比如当成本下降了不超过0.001时，这时候我们就可以认为这一组(w,b)是一对能很好拟合线性模型的值。

4.3 学习率

从梯度下降的式子里我们可以看到有一个α值。

w=w−α∂J∂wb=b−α∂J∂bw = w-\alpha \frac{\partial J}{\partial w} \\ b = b-\alpha \frac{\partial J}{\partial b} w=w−α∂w∂J​b=b−α∂b∂J​

这里的α称为学习率(learning rate)。

他有什么用呢？让我们先回到F(x) = wx的成本函数图像。我们前一节已经讨论过要想是的w1的成本降低，那么w1的值应该要增大。

通过斜率我们确定了w1变化的方向， 那么到底变化多大呢？

如果变化太小，那么w1可能需要更多的梯度下降步骤才能到达最低点

如果变化太大，那么w1可能会错过最低点，并且越来越大，导致无法收敛。

(从函数图像中能看到当成本值越大是，斜率的绝对值也越大，所以相同学习率时成本越大的点会变化的越大)。

所以，学习率是用来控制梯度下降变化幅度的。

当学习率过大时，梯度下降可能会失效找不到最低点；当学习率过小时，又需要更多的步骤才能到达最低点，所以选择一个合适的学习率对梯度下降来说是很重要的。

这里我们只引出概念，关于如何选择学习率，我们下一文再细说。

4.4 线性回归梯度下降公式推导

前面我们得出了

w=w−α∂J∂wb=b−α∂J∂bw = w-\alpha \frac{\partial J}{\partial w} \\ b = b-\alpha \frac{\partial J}{\partial b} w=w−α∂w∂J​b=b−α∂b∂J​

接下来我们把成本函数代入进行推导

w=w−α∂J∂wJ(w,b)=12m∑i=1m(wxi+b−yi)2w=w−12m∑i=1m(wxi+b−yi)22xi=w−1m∑i=1m(wxi+b−yi)2xiw = w-\alpha \frac{\partial J}{\partial w} \\ J(w,b)=\frac{1}{2m} \sum_{i=1}^{m}(wx^i+b-y^i)^2 \\ w = w-\frac{1}{2m} \sum_{i=1}^{m}(wx^i+b-y^i)^22x^i=w-\frac{1}{m}\sum_{i=1}^{m}(wx^i+b-y^i)^2x^i w=w−α∂w∂J​J(w,b)=2m1​i=1∑m​(wxi+b−yi)2w=w−2m1​i=1∑m​(wxi+b−yi)22xi=w−m1​i=1∑m​(wxi+b−yi)2xi

同理能推出

b=b−1m∑i=1m(wxi+b−yi)2b=b-\frac{1}{m}\sum_{i=1}^{m}(wx^i+b-y^i)^2 b=b−m1​i=1∑m​(wxi+b−yi)2

从这里就知道为什么我们前面要在成本函数的前面除以2了， 是为了方便这里的计算。

所以最终线性回归的梯度下降公式就为
