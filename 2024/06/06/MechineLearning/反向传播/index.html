反向传播

反向传播&自动微分是神经网络训练中一个重要的步骤，他帮助我们无需手动计算即可使用梯度下降或Adam优化器来训练神经网络。即使TensorFlow和PyTorch这一类的深度学习框架已经为我们实现了反向传播让我们无需关注， 但是理解反向传播对理解训练过程中的梯度消失梯度爆炸有着重要作用， 理解反向传播的原理也能加深我们对神经网络训练的认识。

一、概念

反向传播是一种用于神经网络中优化算法的方法， 主要用于训练监督学习算法。 反向传播的目的是为了减小神经网络的预测值和真实值之间的误差， 其核心是计算损失函数对于每一个权重的梯度， 这些梯度提供了调整权重的方向和大小。

这个概念一眼看上去可能是一头雾水， 没关系我们后面从由简入深来说明。

在开始之前我们先回顾一下什么是梯度。在线性回归中我们采用梯度下降算法来优化模型， 减小成本。在每一次训练中，我们用成本函数对权重w和偏置项b分别求偏导， 这个偏导就是梯度。即

∂J∂w∂J∂b\frac{\partial J}{\partial w} \\ \frac{\partial J}{\partial b} ∂w∂J​∂b∂J​

这就是梯度，得到了梯度之后， 结合学习率我们就可以执行梯度下降算法来更新权重和偏置项。

w=w−α∂J∂wb=b−α∂J∂bw = w-\alpha \frac{\partial J}{\partial w} \\ b = b-\alpha \frac{\partial J}{\partial b} w=w−α∂w∂J​b=b−α∂b∂J​

反向传播就是在神经网络的优化中自动帮我们计算梯度， 因为梯度是在求偏导。而在反向传播的过程中，复杂的函数求导会比较困难，自动微分采用链式法则将复杂的函数分解为基本操作，使得可以高效的计算导数。

二、反向传播的步骤

反向传播通常会有三个步骤:

  1. 向前传播

    这一步就是在走一遍神经网络， 将数据输入到神经网络中，数据通过每一层传递最后到达输出层，得到神经网络的预测结果。

  2. 计算损失

    在得到神经网络的预测结果之后， 我们就可以将预测结果和真实数据带入到损失函数中计算损失。

  3. 反向传播

    最后根据损失， 利用自动微分技术向后传播， 计算损失对于每一个权重的梯度。最后更新这些权重。

三、简单的反向传播

前面说了我们会由简入深，那我们现在就用一个最简单的神经网络来走一遍反向传播。

这是一个只有一层输出层的神经网络，激活函数是Linear， 即一个最简单的线性回归。 输入只有一个， 输出也只有一个。

x=−2    y=2x=-2\ \ \ \ y=2 x=−2    y=2

那第一步就是向前转播， 将数据输入到神经网络中， 传递后到达输出层， 即

c=wx=−2∗2=−4a=c+b=−4+8=4c = wx = -2*2 = -4\\ a = c+b = -4 + 8 =4 c=wx=−2∗2=−4a=c+b=−4+8=4

得到预测数据后， 我们要计算损失， 即

J=12(a−y)2=2J = \frac{1}{2}(a-y)^2 = 2 J=21​(a−y)2=2

接着我们就开始进行反向传播计算梯度了， 注意我们的目标是要计算损失J对于w和b的梯度，即

∂J∂w∂J∂b\frac{\partial J}{\partial w} \\ \frac{\partial J}{\partial b} ∂w∂J​∂b∂J​

为了计算这两个梯度值， 我们需要利用链式法则。 首先计算J对于a的导数

∂J∂a=a−y=2\frac{\partial J}{\partial a} = a - y =2 ∂a∂J​=a−y=2

接着便可利用链式法则计算J对于c和b的偏导数

∂J∂c=∂J∂a⋅∂a∂c=2∗1=2∂J∂b=∂J∂a⋅∂a∂b=2∗1=2\frac{\partial J}{\partial c} = \frac{\partial J}{\partial a}\cdot\frac{\partial a}{\partial c} =2*1 = 2\\ \frac{\partial J}{\partial b} = \frac{\partial J}{\partial a}\cdot\frac{\partial a}{\partial b} =2*1 = 2\\ ∂c∂J​=∂a∂J​⋅∂c∂a​=2∗1=2∂b∂J​=∂a∂J​⋅∂b∂a​=2∗1=2

接着也能计算出对于w的偏导数

∂J∂w=∂J∂c⋅∂c∂w=2∗(−2)=−4\frac{\partial J}{\partial w} = \frac{\partial J}{\partial c}\cdot\frac{\partial c}{\partial w} =2*(-2) = -4\\ ∂w∂J​=∂c∂J​⋅∂w∂c​=2∗(−2)=−4

我们之所以将wx+b 分成了c=wx 和 a=c+b， 是为了利用链式法则将求导化为最基础的计算。

这样我们就得到了J对于w和b的梯度， 再结合学习率α就可以对w和b进行梯度下降或Adam更新了。

怎么样， 是不是跟着走了一遍之后反向传播的流程就清晰了很多。
