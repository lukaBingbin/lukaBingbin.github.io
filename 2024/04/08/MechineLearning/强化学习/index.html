强化学习

强化学习

一、什么是强化学习

强化学习是通过控制一个Agent进行一些行为(Action)从而改变状态(State)，然后在行动中获得回报(奖励或惩罚)，通过这些回报能学习出对未来的最优指导。

听起来有点抽象，我们举一个例子说明。

假设你正在玩一个类似大富翁的游戏。我们取出其中的6个格子。你的角色在其中的第四个格子。

这里你的角色在第几个格子，就代表了角色的状态(State)。比如此时s=4.

在第一个格子中有100金币， 而在第六个格子中有40金币， 其他格子中都没有钱。

如果角色走到了第一个格子，就能获得这100金币。同理走到第6个格子，就能获得40金币。

每回合你可以选择往右走或者往左走一步。这是行为(Action).

也就是说， 你可以选择连续三个回合往左走， 这样你可以获得这100金币；你也可以选择连续两个回合往右走，你可以获得40金币。

获得的钱，就是行动的回报(return)。

当然你的走法不止单纯向右走或者向左走， 你可以以⬅️➡️➡️⬅️⬅️⬅️⬅️的路线到达1号格子获得100金币，但这显然不是一个很好的路线，因为他需要更多的回合数，付出更大的成本。

而强化学习的目的，就是搞清楚在当前状态下，进行什么样的行为能够使回报最大化，获得最大的收益。

有了这样的目的，强化学习就需要通过学习来达到这样的目的，也就是开头给出的定义。

强化学习的应用非常广泛，比如利用强化学习来自动控制无人机的飞行。强化学习的目的就是要根据无人机每一时刻的状态(位置，旋转角度，飞行速度等)， 判断做出什么操作(向前飞、向后飞、加速、降落等)能够使无人机更平稳地到达目的地。还可以用来控制机器狗爬行，也可以用在游戏领域。

强化学习有三个重要的概念

  • 状态State: 当前Agent的状态。在大富翁的例子中是角色所在的格子，在无人机的例子中是无人机的位置、旋转角度、飞行速度等。
  • 行为Action: 下一步的操作。在大富翁的例子中是向左或向右走，在无人机的例子中是向前飞、向后飞、加速、降落等。
  • 回报Return: 在当前状态下执行行为后的回报。比如在大富翁的例子中向右走两个可以获得40金币。

现在，对强化学习应该有一定的概念了。。

二、强化学习的回报

在大富翁的例子中，当角色处于4号格子时，可以一直往左走或者一直往右走来获取金币。

1号格子的金币虽然比较多，但是要走的格子也更多；6号格子的金币虽然少，但是更容易达到。我们需要在这其中进行取舍，假设现在上面200格的格子处有200个金币， 金币虽然多，但是路途遥远， 显然不如近处的金币更有性价比。

所以，强化学习中获得的回报是会随着行动次数的增加而衰减的。

为了更好的描述回报，我们首先引入一个函数R(s)表示状态s的回报。在大富翁的例子中有

R(1)=100R(6)=40R(2)=0R(1)=100\\ R(6)=40\\ R(2)=0 R(1)=100R(6)=40R(2)=0

然后我们又引入了一个参数γ。γ是一个小于1的数，表示进行一次行动，得到的回报就会降低一些。

从第4格开始一直向左走走到1号格子， 那么回报就应该是

Return=R(4)+γR(3)+γ2R(2)+γ3R(1)Return=R(4)+\gamma R(3) + \gamma^2 R(2) + \gamma^3 R(1)\\ Return=R(4)+γR(3)+γ2R(2)+γ3R(1)

而一直向右走走到6号格子，回报就是

Return=R(4)+γR(5)+γ2R(6)Return=R(4)+\gamma R(5) + \gamma^2 R(6) Return=R(4)+γR(5)+γ2R(6)

通常γ是一个比较接近1的数，例如0.9， 0.99。但是这里为了更明显的理解， 我们令γ=0.5。

代入就可以得到， 当你在第四格的时候， 一直往左走得到的回报是

Return=0+0.5∗0+0.52∗0+0.53∗100=12.5Return=0+0.5*0+0.5^2*0+0.5^3*100 = 12.5 Return=0+0.5∗0+0.52∗0+0.53∗100=12.5

一直往右走得到的回报是

Return=0+0.5∗0+0.52∗40=10Return=0+0.5*0+0.5^2*40=10 Return=0+0.5∗0+0.52∗40=10

同理，我们也可以算出来当你处于其他格子时候的不同方向的回报

如图所示，2、3、4、5号格子左上角的数字表示一直往左走能得到的回报， 右上角的数字表示一直往右走能得到的回报。

当角色处于1和6号格子的时候，无需行动就可以直接获得金币。

三、做出决策

我们已经知道如何计算一组行动能获得的回报了。那么，现在你的角色在第4格内，你应该知道做出哪一个行为能让你获得更多的回报，向左走最终能获得的反馈是12.5，向右走的反馈只有10.

我们定义强化学习中的policy函数，用π来表示，这个函数表示当你处于状态s时，为了获取最大回报应该执行的行为。即

π(s)=a\pi(s) =a π(s)=a

所以有

π(4)=⬅®π(2)=⬅®π(3)=⬅®π(5)=➡®\pi(4) = ⬅️\\ \pi(2)=⬅️\\ \pi(3)=⬅️\\ \pi(5)=➡️ π(4)=⬅R◯π(2)=⬅R◯π(3)=⬅R◯π(5)=➡R◯

四、状态-动作值函数

1、定义

回顾一下强化学习的目的，为了找出处于状态s时使回报最大的行为a。

在大富翁的例子中，当你的角色在4号格子时，你可以选择向左或者向右移动。

  • 如果你向左移动，来到第3格。因为第3格向左移动的回报大于向右移动，所以你会继续向左移动，此时第3格一直向左移动的回报为25， 所以在第4格向左移动的回报=0+γ*25=12.5
  • 如果你向右移动，来到第5格。因为第5格向右移动的回报大于向左移动，所以你会继续向右移动。此时第5格向右移动的回报为20， 所以在第4格向右移动的回报=0+γ*20=10.

对比了两个行为的回报，我们当然会选择回报更高的选择，也就是向左移动。

上面描述的尝试执行一个行为计算回报这个过程，就是状态-动作值函数(state-action value function)。

状态-行为值函数我们用Q(s, a)表示，他执行的操作如下:

  1. 当你处于状态s时
  2. 执行行为a，来到下一个状态s’
  3. 之后执行在状态s‘下最优的行为。

代入前面的例子Q(4, ⬅️)

  1. 当前在第4格内
  2. 向左移动一格，来到第3格
  3. 之后执行第3格的最优行为

在执行完所有可能的操作后，对不同操作的Q(s,a)做比较，选出回报最大的那个行为，就是最优的行为。

你可能会有点疑惑，为了计算Q(s, a), 我们得知道s‘的最优行为的回报，那不是绕回去了吗。 如果都已经知道s的最优行为的回报了， 那也不需要去计算Q(s,a)了。

这是个好问题，这我们会在后面神经网络的训练中去解释。

2、贝尔曼等式

为了更好的描述Q函数，我们有下面这个叫做贝尔曼等式的等式

Q(s,a)=R(S)+γ∗max⁡a′Q(s′,a′)Q(s, a)= R(S)+\gamma * \max_{a'}Q(s', a') Q(s,a)=R(S)+γ∗a′max​Q(s′,a′)

其中s’是由状态s执行了行为a后得到的状态， 而a’是在s’状态下最优的行为。

那我们用贝尔曼等式来表示前面的例子，当你的角色在第4格的时候，

  • 选择向左移动，则有

    Q(4,⬅®)=R(4)+γmax⁡a′Q(3,a′)Q(4,⬅️) =R(4) + \gamma \max_{a'}Q(3, a') Q(4,⬅R◯)=R(4)+γa′max​Q(3,a′)

    这里我们知道在第3格向左的行为回报更高，所以这里

    Q(4,⬅®)=R(4)+0.5∗25=12.5Q(4, ⬅️)=R(4)+0.5 * 25 =12.5 Q(4,⬅R◯)=R(4)+0.5∗25=12.5

  • 选择向右移动，则有

    Q(4,➡®)=R(4)+γmax⁡a′Q(5,a′)Q(4, ➡️) =R(4) + \gamma \max_{a'}Q(5, a') Q(4,➡R◯)=R(4)+γa′max​Q(5,a′)

    我们知道在第5格向右的行为回报最高，所以有

    Q(4,➡®)=R(4)+0.5∗20=10Q(4, ➡️) =R(4) + 0.5*20 = 10 Q(4,➡R◯)=R(4)+0.5∗20=10

有了Q(4, ⬅️)和Q(4, ➡️)的值，我们就可以选择回报更高的那个Q的行为了。

五、连续状态

1、连续状态的实例

前面的大富翁例子，我们的状态是离散的。也就是只有六个状态。

而在实际问题中，我们的状态通常是连续的。让我们换一个例子来说明。我们现在要在游戏里控制一个飞行器安全降落在两个旗帜中间。

此时飞行器的状态有:

state=[xyx˙y˙θθ˙lr]state=\begin{bmatrix} x\\ y\\ \dot{x}\\ \dot{y}\\ \theta\\ \dot{\theta}\\ l\\ r \end{bmatrix} state=​xyx˙y˙​θθ˙lr​​

其中，x为横坐标，y为纵坐标，x’为横坐标上的速度，y‘为纵坐标上的速度，θ是飞行机倾斜的角度，θ’为倾斜的速度，l表示左边是否着陆，r表示右边是否着陆。

在这一组状态里，除了l和r是二元的，其他都是连续的值。

然后我们也可以定义飞行器可执行的行为:

action=[donothingleftthrustermainthrusterrightthruster]action=\begin{bmatrix} do& nothing\\ left&thruster\\ main&thruster\\ right&thruster \end{bmatrix} action=​doleftmainright​nothingthrusterthrusterthruster​​

  • 可以什么都不做，让飞行器自由落体
  • 可以发射左喷射器，这将使飞行器向右旋转
  • 可以发射主喷射器，在飞行器的下方，可以用来减速降落或加速朝一个方向推动
  • 可以发射右喷射器，这将使飞行器向左旋转

有了状态和行为，我们还需要定义反馈：

  • 着陆到制定范围内: +100
  • 坠毁: -100
  • 软着陆: +100
  • 每条腿着地: +10
  • 使用一次主喷射器: -0.3
  • 使用一次左/右喷射器: -0.03

可以看到，我们不光有正向的反馈， 为了节约资源，我们还定义了一些惩罚，让飞行器尽可能少的使用喷射器。

2、构建神经网络学习

在前面我们定义了飞行器的状态，行为和回报。现在我们希望能够计算Q(s,a)得到回报，从而选择能获得最大回报的行为。

我们可以通过构建神经网络来进行预测Q(s,a)的值。

我们的输入为s和a， 所以我们可以把s和a融入到同一个向量X中:

X=[xyx˙y˙θθ˙lr1000]X = \begin{bmatrix} x\\ y\\ \dot{x}\\ \dot{y}\\ \theta\\ \dot{\theta}\\ l\\ r\\ 1\\ 0\\ 0\\ 0 \end{bmatrix} X=​xyx˙y˙​θθ˙lr1000​​

X由s和a组成，其中最后四个数字代表了执行哪一种行为。 例如上面的式子中表示什么都不做。

然后我们构建一个神经网络用来预测Q(s,a)的值。

构建神经网络的目的，是为了通过神经网络预测Q(s,a)的值，所以神经网络的输出y要尽可能接近Q(s,a)

y≈Q(s,a)y\approx Q(s,a) y≈Q(s,a)

训练好神经网络之后，我们就可以计算Q(s, nothing), Q(s, left), Q(s, right), Q(s, main)的值， 然后选择Q(s,a)最大的那个行为a。

注意，我们构建的神经网络不是在状态s的时候预测a， 而是在状态s下执行a后的状态-动作值函数Q(s,a)。我们把这个神经网络叫做Q网络。

3、训练Q网络

构建好Q网络后，我们就需要对Q网络进行训练了。训练需要数据，数据哪里来呢？

数据可以通过实际去操作来获取。比如在状态s下，我们实际去执行一个行为a，进入状态s’。我们能得到这样一组数据:

(s,a,R(s),s′)(s, a, R(s), s') (s,a,R(s),s′)

我们可以执行1000次甚至10000次，获得一组数据:

(s1,a1,R(s1),s′1)(s2,a2,R(s2),s′2)...(s10000,a10000,R(s10000),s′10000)(s^1, a^1, R(s^1), s'^1)\\ (s^2, a^2, R(s^2), s'^2)\\ ...\\ (s^{10000}, a^{10000}, R(s^{10000}), s'^{10000}) (s1,a1,R(s1),s′1)(s2,a2,R(s2),s′2)...(s10000,a10000,R(s10000),s′10000)

有了这些数据，我们就可以利用贝尔曼等式来构造训练数据了。

其中，s和a能组成输入X， R(s)和s’能通过贝尔曼等式来构造出y

y=R(s)+γ∗max⁡a′Q(s′,a′)y = R(s) + \gamma*\max_{a'}Q(s', a') y=R(s)+γ∗a′max​Q(s′,a′)

这样我们就能获得10000组(x,y)的训练数据。这个过程，我们称作经验回放(experience replay)，也就是用过往的经验作为训练的数据。

这时候的你肯定又会有疑问了！要构造y，我哪知道哪个行为a’能使状态s’下的回报最大啊？

别忘了，我们此时已经构建出了Q网络，只是还没训练。我们可以利用现在的Q网络去计算在状态s’下所有行为a’的Q(s’,a’),然后选择最大的那个。 虽然此时Q网络还没经过训练，给出的结果不一定正确，但不要紧， 我们先用他来计算y。

有了训练数据之后，就可以开始训练了。

这里可能有点难以理解，让我们用大富翁的例子再来解释一下。

这是大富翁的格子， 其中回报有所变化。我们构造了一个Q网络，进行随机初始化了。对于此时的Q网络，可能有

Q(3,➡)=20Q(3,⬅)=40Q(4,⬅)=30Q(4,➡)=40Q(3, ➡) = 20\\ Q(3, ⬅) = 40\\ Q(4, ⬅) = 30\\ Q(4, ➡) = 40 Q(3,➡)=20Q(3,⬅)=40Q(4,⬅)=30Q(4,➡)=40

可以看到此时的Q网络预测其实并不准。接着我们来构造训练数据，比如我们要构造在第4个格子向左走的数据，即

(4,⬅,R(4),3)(4, ⬅, R(4), 3) (4,⬅,R(4),3)

那么此时的X = (4, ⬅)。 因为在现在的网络看来，Q(3,⬅)的回报比Q(3, ➡)的回报高，所以此时的

y=R(4)+γ∗Q(4,⬅)=5+0.5∗40=25y = R(4) + γ*Q(4, ⬅) = 5+ 0.5*40 = 25 y=R(4)+γ∗Q(4,⬅)=5+0.5∗40=25

我们就得到了一组数据， 这一组数据我们用Q_new来表示， 即

Qnew(4,⬅)=25Q_{new}(4, ⬅) = 25 Qnew​(4,⬅)=25

现在我们有Q_new(4,⬅)和Q(4, ⬅)， 其中Q_new(4,⬅)是训练数据 所以是y，而Q(4,⬅)是目前Q网络的预测，所以是y^.

Qnew(4,⬅)=yQ(4,⬅)=y^Q_{new}(4, ⬅)= y\\ Q(4, ⬅) = \hat{y} Qnew​(4,⬅)=yQ(4,⬅)=y^​

有了实际值和预测值，我们就可以利用均方误差作为损失函数来更新Q网络的参数，得到一个新的Q网络。所以现在你应该能理解状态-动作值函数的定义了。

然后再继续训练， 事实证明，这样的训练下Q网络的预测值会越来越接近真实的Q(s,a)。

4、改进Q网络

在前面的Q网络中，我们的输入里包含了执行的行为，然后得到Q(s,a)的预测。我们需要对所有可能的行为a进行预测，然后从中选出预测值最大的那个行为。这就意味着，我们必须要预测多次才能得到最佳的行为a。

所以我们对Q网络进行一下改进，将输入只包含状态s, 同时输出层改为n个神经元，也就是输出一个维度为n的向量。例如在飞行器降落的例子中，我们一共有4种行为可选，那么就把Q网络的输出层改为四个神经元。

这样，一次预测我们就能得到所有行为a的Q(s,a)， 大大提高了效率。

5、ε-greedy策略

在飞行器降落的例子种，你可能会发现一个问题。那就是喷射主喷射器的回报是一种惩罚，并且是所有行为种惩罚最大。

这时候你在训练Q网络的过程中， 因为贝尔曼等式， 在计算Q(s,a)的时候，几乎永远不会选择发动主喷射器作为a’， 因为这不满足a’使得Q(s’, a’)最大。

然而，在一些情况下（例如下降速度过快）选择主喷射器是是一个比较好的选择。

所以，我们可以不总是选择使Q(s,a)最大的行为a， 所以在少数情况下随机选择一个行为。

我们定义一个参数ε，例如ε=0.05， 那么在95%的情况下我们都选择令Q值最大的行为a， 而在5%的情况下我们随机选择一个行为a。

  • 我们将选择令Q值最大的行为a称作利用， 因为他会选择得到已知最大回报的动作，这时候他也是贪心的。

  • 把随机选择行为a称作探索，在这种情况下，算法有可能会发现更好的策略。
