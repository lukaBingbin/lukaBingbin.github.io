4.1 线性回归

y^=θ⋅x\hat{y} = \theta \cdot x y^​=θ⋅x

其中\hat{y}是预测值

n是特征数量

X是特征值(X_0, X_1, X_2, …, X_n, 其中X_0始终为1)

θ 是模型特征 θ_0是偏置项, 其余是特征权重

成本函数MSE

MSE=12m∑i=1m(θ⋅xi−yi)2=12m(Xθ−y)T(Xθ−y)MSE = \frac{1}{2m}\sum_{i=1}^m(\theta\cdot x^i - y^i)^2 = \frac{1}{2m}(X\theta-y)^T(X\theta-y) MSE=2m1​i=1∑m​(θ⋅xi−yi)2=2m1​(Xθ−y)T(Xθ−y)

为了得到使得MSE最小的θ值， 我们可以用标准方程和梯度下降两种方法

4.1.1 标准方程

θ^=(XTX)−1XTy\hat{\theta} = (X^TX)^{-1}X^Ty θ^=(XTX)−1XTy

在这个方差中， \hat{θ}是使得成本函数最小的θ值。

y是y^1到y^m的目标值向量

这个方程可以由成本函数MSE推导而来
我们假设X^TX 是可逆的，

MSE=12m(Xθ−y)T(Xθ−y)MSE = \frac{1}{2m}(X\theta-y)^T(X\theta-y) MSE=2m1​(Xθ−y)T(Xθ−y)

对这个函数展开

MSE=12m(θTXXθ−θTXTY−YTXθ−YTY)MSE = \frac{1}{2m}(\theta^TX^X\theta - \theta^TX^TY - Y^TX\theta - Y^TY) MSE=2m1​(θTXXθ−θTXTY−YTXθ−YTY)

然后因为

θTXTY是一个标量,  θTXTY=YTXθ\theta^TX^TY是一个标量, \ \ \\ \theta^TX^TY = Y^TX\theta θTXTY是一个标量,  θTXTY=YTXθ

所以有

MSE=12m(θTXXθ−2YTXθ−YTY)MSE = \frac{1}{2m} (\theta^TX^X\theta - 2Y^TX\theta - Y^TY) MSE=2m1​(θTXXθ−2YTXθ−YTY)

接着我们对theta求偏导

∂MSE∂θ=12m(2XTXθ−2XTY)\frac{\partial{MSE}}{\partial\theta} = \frac{1}{2m}(2X^TX\theta - 2X^TY) ∂θ∂MSE​=2m1​(2XTXθ−2XTY)

要找到令MSE最小的θ， 那我们令导数=0

2XTXθ−2XTY=02X^TX\theta - 2X^TY = 0 2XTXθ−2XTY=0

即

θ^=(XTX)−1XTy\hat{\theta} = (X^TX)^{-1}X^Ty θ^=(XTX)−1XTy

4.1.2 计算复杂度

因为标准方程要计算X^TX的逆， 是一个(n+1)x(n+1)的矩阵， 这种矩阵求逆的时间复杂度通常为O(n²)到O(n³)

如果特征值翻倍，那么计算时间会大概增加四倍

所以在特征值比较多的时候， 标准方程就太慢了。

4.2.2 随机梯度下降

批量梯度下降有一个缺点，因为每次训练都是用整个训练数据集， 所以训练的速度会变慢。

而随机梯度下降是随机选择一个实例来计算梯度， 每次下降仅根据这一个实例来计算。运算速度就会快很多。

当然随机梯度下降就会很随机， 在梯度下降的过程中就不是一直朝着最小值去， 而是上上下下弯弯曲曲， 但是整体还是在下降的。

而且随机梯度下降就算到了最小值， 也不会停下来。所以随机梯度下降得到的值是比较好的，但不是最优的。

随机梯度下降还有一个优势在于他能逃离局部最优值。当成本函数不是一个凹函数的时候， 随机梯度下降更有可能找到全局最小值。

也就是说随机梯度下降能逃离局部最优值， 但是找不到最小值。 解决的办法是在下降过程中调整学习率， 在一开始学习率比较大， 下降的步长比较长；而在后面学习率慢慢减小， 步长减小， 变化不会很大， 有助于我们找到全局的最小值。

4.2.3 小批量梯度下降

小批量梯度下降就是在批量梯度下降和随机梯度下降中间，他不会用整个训练数据集计算梯度， 也不是随机抽一个实例， 而且取整个训练数据集的一部分来计算梯度。

好处在于计算会比批量梯度下降要快， 对于凹函数来说得到的最小值也比随机梯度下降更优。

缺点在于成本函数非凹函数的话， 也很难逃脱局部最小值的问题。

算法 m很大 核外支持 n很大 超参数 要求缩放 Scikit-Learn
标准方程 快 否 慢 0 否 N/A
SVD 快 否 慢 0 否 LinearRegression
批量GD 慢 否 快 2 是 SGDRegressor
随机GS 快 是 快 >=2 是 SGDRegressor
小批量GD 快 是 快 >=2 是 SGDRegressor

4.5 正则化线性模型

解决过度拟合的一个方法就是对线性模型进行正则化。即减少模型的自由度。

4.5.1 岭回归

岭回归 也成为TikHonov正则化。即在成本函数的后面加上一段

J(θ)=MSE(θ)+λ12∑i=1nθi2J(\theta) = MSE(\theta) + \lambda\frac{1}{2}\sum_{i=1}^n\theta_i^2 J(θ)=MSE(θ)+λ21​i=1∑n​θi2​

可见过度拟合

4.5.2 Lasso回归

和岭回归类似， Lasso回归也是在成本函数后面加上一个正则项。

J(θ)=MSE(θ)+λ∑i=1n∣θ∣J(\theta) = MSE(\theta) + \lambda\sum_{i=1}^n|\theta| J(θ)=MSE(θ)+λi=1∑n​∣θ∣

岭回归是降低不重要特征的权重(很低，还不会为0)， 而Lasso回归更倾向于将不重要的特征值权重完全消除（即设置为0）。

4.5.3 弹性回归

弹性回归位于岭回归和Lasso回归之间

J(θ)=MSE(θ)+rλ∑i=1n∣θ∣+1−r2λ∑i=1nθi2J(\theta) = MSE(\theta) + r\lambda\sum_{i=1}^n|\theta| + \frac{1-r}{2}\lambda\sum_{i=1}^n\theta_i^2 J(θ)=MSE(θ)+rλi=1∑n​∣θ∣+21−r​λi=1∑n​θi2​

通常对于线性回归，有正则化总比没有好。 岭回归可以作为默认的回归方法， 而弹性回归的表现要比Lasso回归好。 当你觉得需要的特征值只有几个的时候， 也可以考虑弹性回归。 通过控制r的值，就可以让弹性回归更偏向岭回归或者Lasso回归。

4.5.4 提前停止

对于梯度下降这一类迭代学习的算法， 还有一个特殊的正则化方法：提前停止。

即在训练时， 让交叉验证集的误差到达最小值的时候，就停止训练。

因为当模型过拟合的时候，在交叉验证集上的误差会增大。
